# Contributing to Tax Intelligence

We welcome contributions to the Tax Intelligence platform! Whether it's bug reports, feature requests, or code contributions, we value your feedback and help.

## How to Contribute

1. **Fork the repository**: Click the "Fork" button on the top right of this repository to create your own copy.
2. **Clone your fork**: Clone the repository to your local machine.
3. **Create a branch**: Create a new branch for your feature or bug fix (`git checkout -b feature/your-feature-name`).
4. **Make your changes**: Write your code, update documentation, and ensure everything runs locally.
5. **Commit your changes**: Write clear, concise commit messages.
6. **Push to your branch**: Push your changes to your fork on GitHub.
7. **Submit a Pull Request**: Go to the original repository and submit a Pull Request describing your changes.

## Development Setup

Please refer to the `README.md` for instructions on setting up the local environment, including virtual environments and dependencies.

## Code Style

- We follow standard Python PEP 8 guidelines.
- Please ensure all new logic is well-commented.
- Add type hints where applicable.

Thank you for helping us improve Tax Intelligence!
